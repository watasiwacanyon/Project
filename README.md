# 🚨 Центр мониторинга киберугроз

## 📌 Описание
Интерактивный дашборд для визуализации и анализа киберугроз в реальном времени.  
Разработан для участия в конкурсе.

## 🛠 Технологии
- **Python 3.13** — язык программирования
- **Streamlit** — фреймворк для веб-интерфейса
- **Plotly** — интерактивные графики и карта
- **Pandas** — обработка данных
- **Faker** — генерация тестовых данных

## 🚀 Запуск
```bash
pip install -r requirements.txt
streamlit run app.py
